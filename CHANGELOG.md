# Changelog

## [1.14.0] - 2026-01-19

### Added
- **Route Visualization**: Color-coded routes by speed, power consumption, or elevation
- Full Turkish language support (Türkçe)

### Fixed
- Bug fixes and performance improvements

## [1.13.0] - 2026-01-07

### Added
- **Home Screen Widgets**: Battery, range, and location at a glance
- **Timeline View**: Parking history and vampire drain analysis
- **Battery Health**: Manual 'new battery' capacity setting for accurate health calculations
- Full German language localization

## [1.12.5] - 2025-12-22

### Fixed
- Several community-reported bugs
- Improved app responsiveness and data loading speed
- Enhanced overall stability

## [1.12.4] - 2025-12-17

### Added
- Complete Italian language localization

### Fixed
- Various bug fixes and improvements

## [1.12.3] - 2025-12-15

### Added
- **Rewind 2025**: Personalized yearly driving statistics in a shareable format

### Fixed
- Pro license sync with multiple Apple IDs

## [1.12.2] - 2025-12-03

### Fixed
- More accurate energy consumption in Trips
- Improved efficiency calculations for short drives
- Better handling of preconditioning energy

## [1.12.1] - 2025-12-01

### Added
- **Trips Feature**: Group drives and charging sessions into meaningful journeys
- Edit charging costs from the charging detail screen
- New languages: Simplified Chinese and Korean

### Changed
- Cleaner interface during DC fast charging

## [1.11.1] - 2025-11-09

### Fixed
- Sync issue for new drives and charges
- Improved data refresh reliability

## [1.11.0] - 2025-11-08

### Fixed
- Accurate maps in China with GCJ-02 coordinate fix

### Changed
- Filter settings saved between app sessions
- Performance and stability improvements

## [1.10.0] - 2025-11-02

### Added
- **Battery Health**: Tracking and degradation monitoring
- Display charging costs in your preferred currency
- Filter out short drives with minimum distance setting
- Total energy consumption shown in drive details

### Changed
- Compatible with TeslaMate API 1.23

## [1.9.0] - 2025-10-09

### Added
- Quick vehicle switcher on dashboard and list views
- New Apple Watch complication widget

### Changed
- Enhanced Apple Watch app experience

### Fixed
- Various bug fixes

## [1.8.0] - 2025-10-03

### Added
- New in-app feedback option

### Fixed
- HTTP connection reliability issues
- Various bug fixes

### Changed
- Refined Liquid Glass button animations

## [1.7.0] - 2025-10-01

### Added
- **Apple Watch App**: Quick access to battery, range, and drive info
- Redesigned setup wizard for easier configuration

### Fixed
- Bug fixes and performance improvements
