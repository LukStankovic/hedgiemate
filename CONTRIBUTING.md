# Contributing to HedgieMate

HedgieMate is a closed-source iOS app. This repository contains documentation, issue tracking, and community resources.

## How to Contribute

### Report Bugs

Found a bug? [Open an issue](https://github.com/LukStankovic/hedgiemate/issues/new?template=bug_report.yml) with:

- App and OS version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if helpful

### Suggest Features

Have an idea? [Submit a feature request](https://github.com/LukStankovic/hedgiemate/issues/new?template=feature_request.yml).

Check [existing requests](https://github.com/LukStankovic/hedgiemate/issues?q=is%3Aissue+label%3Aenhancement) first - upvote with a reaction if someone already suggested it.

### Translations

Help make HedgieMate available in more languages on [Crowdin](https://crowdin.com/project/hedgiemate/invite?h=bff10a865074daf9d4e08ef064507e982646816).

Translation contributors receive a **free Pro license**.

### Documentation

Spotted a typo or unclear instructions? PRs to documentation files are welcome.

### Community Support

Help other users on [Discord](https://discord.gg/h6vxuG4ZH8) by answering questions about setup and usage.

## Code of Conduct

Be respectful. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
